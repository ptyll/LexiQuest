## Component Details

### Avatar Upload

```
Before:
┌───────────┐
│           │
│  [Icon]   │
│           │
│   [📷]    │
└───────────┘

After upload:
┌───────────┐
│ ┌───────┐ │
│ │ Photo │ │
│ │       │ │
│ └───────┘ │
│   [📷]    │
└───────────┘
```

#### Upload flow
```
1. Click photo icon
2. Open file picker
3. Preview with crop
4. Confirm upload
5. Progress bar
6. Success checkmark
```

### Toggle Switch

```
Off:    ┌──────────┐    On:    ┌──────────┐
        │     ●    │            │    ●     │
        │░░░░░░    │            │    ██████│
        └──────────┘            └──────────┘
        
Colors:
- Off: --color-gray-300
- On: --color-primary-500
- Handle: white
```
